# Aviso!

O Backend cria automaticamente todas as tabelas no PostgreSQL, basta criar manualmente o banco de dados (não as tabelas)
e trocar as informações de conexão necessárias em backend/config/database.js

Os scripts SQL que estão presentes em /database apenas populam o banco de dados e instalam extensões. É extremamente recomendado que eles sejam executados antes usar o projeto.
